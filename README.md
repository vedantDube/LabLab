## ğŸš€ Getting Started

### Prerequisites

- Python 3.8+
- Node.js 16+
- ChatGPT-5 API Key
- Ethereum wallet (for blockchain features)

### Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/yourusername/carbontwin.git
   cd carbontwin
   ```

2. **Backend Setup**

   ```bash
   cd backend
   pip install -r requirements.txt
   ```

3. **Frontend Setup**

   ```bash
   cd frontend
   npm install
   ```

4. **Environment Configuration**

   ```bash
   cp .env.example .env
   # Add your ChatGPT-5 API key and other configurations
   ```

5. **Start the Application**

   ```bash
   # Backend (Terminal 1)
   cd backend
   python app.py

   # Frontend (Terminal 2)
   cd frontend
   npm start
   ```
